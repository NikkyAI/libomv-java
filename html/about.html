About libomv-java

libomv-java is a Java based Client/Server library used for accessing and creating 3d virtual worlds.

libomv-java is for a large part derived from translating the Open Metaverse C# library to Java.

The library maintains compatibility with the Second Life network protocol and can be used for creating clients and automatons in Second Life, OpenSim or other virtual worlds which use the Second Life network protocol.

The libomv-java library is compatible with the Java SE 6 or newer.
