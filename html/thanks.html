Thanks

The libomv-java project would like to thank our supporters, donors, and all other contributors.

Thanks go to Simon Whiteside, who started with the translation of the libsecondlife library to Java, and built the founding base for this library. The original code can be found under libsecondlife-java Project.

Special thanks go to the Open Metaverse Foundation, whose C# code served as starting point for most of the code in this library.

We would also like to thank the people from TeamDev who were so kind to provide a free license for the use of their JxBrowser component in the GUI part of this project.
